## Build Setup

``` bash
# 安装依赖
npm install

# 本地环境调试，浏览器会自动打开：http://localhost:5000
npm run dev

# 生产环境打包
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
